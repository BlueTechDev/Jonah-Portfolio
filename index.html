Project Overview

This project focuses on developing practical skills in Security Operations Center (SOC) and penetration testing. Key objectives include:

  • Simulating real-world cyber attack scenarios.
  • Mastering tools like Wireshark, Nmap, and Hydra.
  • Implementing defensive measures using UFW and other technologies.
  • Documenting findings and refining incident response strategies.
